## 📋 О проекте

Этот проект представляет собой **графический симулятор машины Тьюринга**, разработанный на Qt/C++. Программа позволяет создавать алгоритмы, визуально наблюдать за работой машины, отслеживать перемещение головки и изменение символов на ленте в реальном времени.

### ✨ Особенности

- 🎨 **Визуальная лента** — каждый символ отображается в отдельной ячейке с рамкой
- 🎬 **Плавная анимация** — головка перемещается с анимацией между шагами
- 🎯 **Подсветка правил** — текущее правило автоматически подсвечивается в таблице
- ⚡ **Регулировка скорости** — ползунок для изменения скорости выполнения
- 📝 **Гибкое управление** — пошаговое выполнение, автоматический запуск, остановка
- 🔤 **Настраиваемый алфавит** — основной и дополнительный алфавиты
- ➕ **Динамические состояния** — возможность добавлять новые состояния

## 🚀 Установка и запуск

### Требования

- Qt 5.15 или новее
- C++17 совместимый компилятор
- CMake 3.10+ (опционально)

### Сборка из исходников

```bash
# Клонирование репозитория
git clone https://github.com/xewhysis/TURING.git
cd TURING

# Сборка с помощью qmake
qmake && make

# Или с помощью CMake
mkdir build && cd build
cmake ..
make

# Запуск
./turing-machine
